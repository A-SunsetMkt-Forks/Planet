## DEVELOPMENT_TEAM

To use your own build config, please create a `local.xcconfig` file alongside the `Planet.xcconfig` file.

The following configuration variables can be overridden:

- DEVELOPMENT_TEAM
- CODE_SIGN_STYLE
- ORGANIZATION_IDENTIFIER_PREFIX
